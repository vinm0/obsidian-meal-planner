# Meal Planner

An Obsidian plugin for planning your meals and generating shopping lists.

## Features

- **Meal Management**: Create and store meals with ingredients, quantities, and cooking instructions
- **Weekly Planning**: Organize meals using a calendar-style weekly planner
- **Shopping Lists**: Automatically generate ingredient summaries for your planned meals

## Installation

### From Obsidian Community Plugins

1. Open Settings in Obsidian
2. Navigate to Community Plugins and disable Safe Mode
3. Click Browse and search for "Meal Planner"
4. Click Install, then Enable

### Manual Installation

1. Download the latest release from the [Releases](https://github.com/yourusername/obsidian-meal-planner/releases) page
2. Extract the files into your vault's `.obsidian/plugins/meal-planner/` folder
3. Reload Obsidian
4. Enable the plugin in Settings → Community Plugins
